TTRPG OWL

TL;DR

An anthology of game designs in tabletop role-playing games. Hosted at https://github.com/pmartinolli/TTRPG_OWL. Built by Pascal Martinolli (2019-now).

Visualizations

«Classes» are basic game ideas.
« Individuals » are actual game designs existing for a specific TTRPG and that can be described with one or several classes.
